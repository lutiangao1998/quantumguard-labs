# 量子安全冷存储方案 (Q-Safe Cold Storage) 技术文档

## 1. 简介

量子安全冷存储方案旨在为超大额资产提供最高等级的物理隔离保护。通过离线 PQC 签名协议，确保私钥在整个迁移过程中永不触网，从而抵御包括量子攻击在内的所有网络威胁。

## 2. 核心组件

### 2.1 离线签名协议 (Offline Signing Protocol)
定义了在线观察钱包与离线签名设备之间的数据交换格式。支持通过二维码（QR Code）或物理介质（如 SD 卡）进行数据传输。

### 2.2 气隙环境支持 (Air-gapped Support)
系统设计确保签名设备无需任何网络连接（Wi-Fi, Bluetooth, Cellular），仅通过视觉或物理接口进行交互。

## 3. 工作流程

1.  **在线准备**：在联网设备上构建迁移交易，并生成待签名负载（Signing Payload）。
2.  **离线传输**：通过扫描二维码将负载传输至离线签名设备。
3.  **离线签名**：在离线设备上使用 PQC 私钥对交易进行签名。
4.  **签名回传**：将生成的 PQC 签名通过二维码回传至联网设备。
5.  **广播交易**：联网设备验证签名后，将抗量子交易广播至区块链网络。

## 4. 安全优势

*   **私钥隔离**：私钥存储在完全离线的环境中，无法被远程攻击。
*   **抗量子算法**：使用 ML-DSA 或 Falcon 算法，确保签名本身具备抗量子能力。
*   **审计追踪**：所有离线操作均可记录并用于后续审计。

## 5. 未来扩展
